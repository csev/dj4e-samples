DJ4E Samples
============

This is a set of projects that provide free sample code to
support the www.dj4e.com web site.  These samples can be used
for Django 4.2 or Django 5.2.  Each is on a branch that you should
checkout.  Here are the instructions for the Django versions:

* Django 4.2 [README.md](https://github.com/csev/dj4e-samples/blob/django52/README.md)

* Django 5.2 [README.md](https://github.com/csev/dj4e-samples/blob/django42/README.md)

You can see some of the samples working at

http://samples.dj4e.com
